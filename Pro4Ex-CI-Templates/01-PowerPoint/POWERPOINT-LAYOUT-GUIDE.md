## 📊 OVERVIEW

Die Pro4Ex PowerPoint-Vorlage enthält **10 professionelle Slide-Layouts** mit konsistentem Corporate Design. Alle Slides haben:

- ✅ **Schwarzer Balken oben** (0.65" hoch) – Premium Corporate Header
- ✅ **Pro4Ex Logo** (0.6" groß, oben rechts) – Markenidentität
- ✅ **Weiße Überschriften** im Balken (24pt, Bold) – Hohe Lesbarkeit
- ✅ **Dark Mode Design** (#0F172A Hintergrund) – Modern & Professional

## 🎯 LAYOUT-ÜBERSICHT

### Slide 1: Title Slide (Cover)

**Zweck:** Titelseite für Präsentationen  
**Inhalt:** Großer Titel + Untertitel + Datum/Autor

**Best Practice:**
- Titel: 48-56pt, weiß, im schwarzen Balken
- Untertitel: 24-28pt, hell-grau, in Content-Area
- Verwende die untere 2/3 der Slide für Content

```
[schwarzer Balken mit Logo]
[TITEL IN GROSS]
[Untertitel – Kleinere Schrift]
[Datum & Präsentant]
```

---

### Slide 2: Chapter/Section Header

**Zweck:** Kapitel-Trennseite (z.B. "1. Supply Chain Strategy")  
**Inhalt:** Kapitelnummer + Titel + Optional: Icon/Bild

**Best Practice:**
- Nur Titel im Balken
- Großes Icon oder abstraktes Bild rechts
- Minimalistisches, fokussiertes Design

```
[schwarzer Balken mit "1. Chapter Title"]
[              Icon/Grafik              ]
[      (rechts ausgerichtet)             ]
```

---

### Slide 3: Content + Text (1-Column)

**Zweck:** Hauptinhalt mit Überschrift + Absätze  
**Inhalt:** Bullet Points oder Fließtext

**Best Practice:**
- Überschrift im Balken (schwarz)
- 3-5 Bullet Points max
- Line-Spacing: 1.5x für Lesbarkeit
- Font: 16-18pt für Body Text

```
[schwarzer Balken mit "Überschrift"]
[• Punkt 1]
[• Punkt 2]
[• Punkt 3]
```

---

### Slide 4: Two-Column (Text + Image)

**Zweck:** Text links, Bild/Grafik rechts  
**Inhalt:** Inhalt + visuelle Unterstützung

**Best Practice:**
- Links: 50% für Text (3-4 Bullet Points)
- Rechts: 50% für Bild (hochwertige Grafik oder Foto)
- Bild-Höhe: Min 2 Zoll, Min 3:2 Ratio

```
[schwarzer Balken]
[• Text links  |  [Bild rechts] ]
[• Text links  |  [Bild rechts] ]
[• Text links  |  [Bild rechts] ]
```

---

### Slide 5: Statistics/Numbers

**Zweck:** Große Zahlen mit Kontext (KPIs, Metriken)  
**Inhalt:** 3-4 große Nummern + Beschreibungen

**Best Practice:**
- Nummern: 72-96pt, Pro4Ex Blue (#0057CC)
- Label: 14-16pt, grau
- Cards in Light-Boxes für bessere Lesbarkeit
- Max 4 Nummern pro Slide (nicht überladen)

```
[schwarzer Balken mit Titel]
[  25%  |   $4.2M  |   98%   ]
[Growth | Revenue  | Uptime  ]
```

---

### Slide 6: 4-Box Grid (Services/Features)

**Zweck:** 4 Items in Gitter (Services, Benefits, Features)  
**Inhalt:** 4 Boxes mit Icon + Titel + Beschreibung

**Best Practice:**
- 2×2 Grid Layout
- Boxes: ~3"×2.5" each
- Icon oben (32×32px)
- Titel: 16pt Bold
- Beschreibung: 12pt, max 2-3 Zeilen
- Box-Farbe: #1E293B (leicht heller als BG)

```
[schwarzer Balken]
[┌─────────┬─────────┐]
[│ Supply  │ Quality │]
[│ Chain   │ Mgmt.   │]
[├─────────┼─────────┤]
[│ Cost    │ Risk    │]
[│ Control │ Mitiga  │]
[└─────────┴─────────┘]
```

---

### Slide 7: Timeline/Process Flow

**Zweck:** Prozessschritte oder zeitliche Abfolge  
**Inhalt:** 5-6 Schritte mit Pfeilen/Verbindungen

**Best Practice:**
- Horizontal oder vertikal angeordnet
- Nummern: 20-24pt, weiß auf blau
- Schritt-Beschreibung: 12-14pt
- Pfeile: Pro4Ex Blue (#0057CC)
- Farbcodierung optional (Status: grün/orange/rot)

```
[schwarzer Balken: "Projektablauf"]
[① Plan → ② Analyze → ③ Execute → ④ Monitor]
```

---

### Slide 8: Comparison Table (Vor/Nach oder Konkurrenz)

**Zweck:** Vergleiche darstellen (alt vs neu, Optionen, Wettbewerb)  
**Inhalt:** 2-3 Spalten, 5-6 Zeilen

**Best Practice:**
- Header: Dark-Blue (#0057CC), weiß Text
- Zeilen abwechselnd: #1E293B und #0F172A
- Text: Links-ausgerichtet, 12-14pt
- Max 6 Zeilen (sonst unlesbar)
- Checkmarks für Vergleiche

```
[schwarzer Balken]
┌──────────┬─────────┬─────────┐
│ Kriterium│ Option 1│ Option 2│
├──────────┼─────────┼─────────┤
│ Kosten   │   ✓✓✓   │   ✓     │
│ Speed    │   ✓     │   ✓✓    │
└──────────┴─────────┴─────────┘
```

---

### Slide 9: Quote/Testimonial

**Zweck:** Kundenzitat oder Motivation  
**Inhalt:** Großes Zitat + Quelle

**Best Practice:**
- Zitat: 32-40pt, italic, Pro4Ex Blue
- Quelle: 14-16pt, grau (Name + Title)
- Hintergrund: Optional subtle-Gradient
- Max 2-3 Sätze (kurz & prägnant)
- Icon: Quotation Mark (") optional links

```
[schwarzer Balken]
["Proficiency for Execution
  ist unser Standard."]
– Jan Michaelsen, Founder
```

---

### Slide 10: Closing/CTA (Call-to-Action)

**Zweck:** Abschlussseite mit Kontaktinformation  
**Inhalt:** Dank + Kontaktdaten + CTA Button

**Best Practice:**
- Zentral ausgerichtet
- Titel: "Thank You" oder "Questions?"
- Kontakt: E-Mail + Telefon + Website
- CTA Button: Prominent (Pro4Ex Blue)
- Optional: Social Media Icons

```
[schwarzer Balken mit Kontakt-Info]
[        Thank You              ]
[  jan.michaelsen@pro4ex.de     ]
[  +49 123 456789               ]
[  www.pro4ex.consulting        ]
[   [VISIT WEBSITE]             ]
```

## 🎨 STYLING REGELN

### Farben

| Element | Farbe | Kontrast |
|---------|-------|----------|
| Balken | #000000 (Schwarz) | N/A |
| Überschrift | #FFFFFF (Weiß) | 21:1 ✓ |
| Text | #CBD5E1 (Helles Grau) | 7:1 ✓ |
| Accent/Buttons | #0057CC (Pro4Ex Blue) | 8.2:1 ✓ |
| BG | #0F172A (Dark Navy) | – |
| Card-BG | #1E293B (Leicht heller) | – |

### Typografie

| Element | Font | Größe | Weight |
|---------|------|-------|--------|
| Balken-Title | Roboto Flex | 24pt | Bold (700) |
| Überschrift (Content) | Roboto Flex | 28-32pt | Bold (700) |
| Body Text | Roboto Flex | 16-18pt | Regular (400) |
| Caption/Label | Roboto Flex | 12-14pt | Regular (400) |
| Numbers/Stats | Roboto Flex | 48-96pt | Bold (700) |

### Spacing

| Element | Padding |
|---------|---------|
| Balken Innen | 8px vertical, 12px horizontal |
| Content Margin | 20px vom Rand |
| Box-zu-Box Spacing | 16px |
| Line-Height | 1.5x (Body), 1.2x (Titles) |

## 🔧 HÄUFIGE ANPASSUNGEN

### Slide-Inhalt ändern
1. Öffne die Slide in PowerPoint
2. Klicke auf Text im Balken
3. Überschrift editieren (Weiß bleibt automatisch)
4. Inhalt darunter anpassen

### Farben anpassen
1. **Balken-Farbe:** Format → Shape Fill → (Farbe)
2. **Text-Farbe:** Text auswählen → Font Color → Weiß (#FFFFFF)
3. **Buttons:** Format → Fill → Pro4Ex Blue (#0057CC)

### Logo-Position ändern
1. Logo rechts oben anklicken
2. Mit Pfeilen positionieren
3. **NICHT**: Größe sollte bei 0.6" bleiben
